2ErrorClass Plugins 

3------------------ 

4 

5ErrorClass plugins provide an easy way to add support for custom 

6handling of particular classes of exceptions. 

7 

8An ErrorClass plugin defines one or more ErrorClasses and how each is 

9handled and reported on. Each error class is stored in a different 

10attribute on the result, and reported separately. Each error class must 

11indicate the exceptions that fall under that class, the label to use 

12for reporting, and whether exceptions of the class should be 

13considered as failures for the whole test run. 

14 

15ErrorClasses use a declarative syntax. Assign an ErrorClass to the 

16attribute you wish to add to the result object, defining the 

17exceptions, label and isfailure attributes. For example, to declare an 

18ErrorClassPlugin that defines TodoErrors (and subclasses of TodoError) 

19as an error class with the label 'TODO' that is considered a failure, 

20do this: 

21 

22 >>> class Todo(Exception): 

23 ... pass 

24 >>> class TodoError(ErrorClassPlugin): 

25 ... todo = ErrorClass(Todo, label='TODO', isfailure=True) 

26 

27The MetaErrorClass metaclass translates the ErrorClass declarations 

28into the tuples used by the error handling and reporting functions in 

29the result. This is an internal format and subject to change; you 

30should always use the declarative syntax for attaching ErrorClasses to 

31an ErrorClass plugin. 

32 

33 >>> TodoError.errorClasses # doctest: +ELLIPSIS 

34 ((<class ...Todo...>, ('todo', 'TODO', True)),) 

35 

36Let's see the plugin in action. First some boilerplate. 

37 

38 >>> import sys 

39 >>> import unittest 

40 >>> try: 

41 ... # 2.7+ 

42 ... from unittest.runner import _WritelnDecorator 

43 ... except ImportError: 

44 ... from unittest import _WritelnDecorator 

45 ...  

46 >>> buf = _WritelnDecorator(sys.stdout) 

47 

48Now define a test case that raises a Todo. 

49 

50 >>> class TestTodo(unittest.TestCase): 

51 ... def runTest(self): 

52 ... raise Todo("I need to test something") 

53 >>> case = TestTodo() 

54 

55Prepare the result using our plugin. Normally this happens during the 

56course of test execution within nose -- you won't be doing this 

57yourself. For the purposes of this testing document, I'm stepping 

58through the internal process of nose so you can see what happens at 

59each step. 

60 

61 >>> plugin = TodoError() 

62 >>> from nose.result import _TextTestResult 

63 >>> result = _TextTestResult(stream=buf, descriptions=0, verbosity=2) 

64 >>> plugin.prepareTestResult(result) 

65 

66Now run the test. TODO is printed. 

67 

68 >>> _ = case(result) # doctest: +ELLIPSIS 

69 runTest (....TestTodo) ... TODO: I need to test something 

70 

71Errors and failures are empty, but todo has our test: 

72 

73 >>> result.errors 

74 [] 

75 >>> result.failures 

76 [] 

77 >>> result.todo # doctest: +ELLIPSIS 

78 [(<....TestTodo testMethod=runTest>, '...Todo: I need to test something\\n')] 

79 >>> result.printErrors() # doctest: +ELLIPSIS 

80 <BLANKLINE> 

81 ====================================================================== 

82 TODO: runTest (....TestTodo) 

83 ---------------------------------------------------------------------- 

84 Traceback (most recent call last): 

85 ... 

86 ...Todo: I need to test something 

87 <BLANKLINE> 

88 

89Since we defined a Todo as a failure, the run was not successful. 

90 

91 >>> result.wasSuccessful() 

92 False

100class MetaErrorClass(type): 

101 """Metaclass for ErrorClassPlugins that allows error classes to be 

102 set up in a declarative manner. 

103 """ 

104 def __init__(self, name, bases, attr): 

105 errorClasses = [] 

106 for name, detail in list(attr.items()): 

107 if isinstance(detail, ErrorClass): 

108 attr.pop(name) 

109 for cls in detail: 

110 errorClasses.append( 

111 (cls, (name, detail.label, detail.isfailure))) 

112 super(MetaErrorClass, self).__init__(name, bases, attr) 

113 self.errorClasses = tuple(errorClasses) 

114 

115 

116class ErrorClass(object): 

117 def __init__(self, *errorClasses, **kw): 

118 self.errorClasses = errorClasses 

119 try: 

120 for key in ('label', 'isfailure'): 

121 setattr(self, key, kw.pop(key)) 

122 except KeyError: 

123 raise TypeError("%r is a required named argument for ErrorClass" 

124 % key) 

125 

126 def __iter__(self): 

127 return iter(self.errorClasses)

130class ErrorClassPlugin(Plugin, metaclass=MetaErrorClass): 

131 """ 

132 Base class for ErrorClass plugins. Subclass this class and declare the 

133 exceptions that you wish to handle as attributes of the subclass. 

134 """ 

135 score = 1000 

136 errorClasses = () 

137 

138 def addError(self, test, err): 

139 err_cls, a, b = err 

140 if not isclass(err_cls): 

141 return 

142 classes = [e[0] for e in self.errorClasses] 

143 if [c for c in classes if issubclass(err_cls, c)]: 

144 return True 

145 

146 def prepareTestResult(self, result): 

147 if not hasattr(result, 'errorClasses'): 

148 self.patchResult(result) 

149 for cls, (storage_attr, label, isfail) in self.errorClasses: 

150 if cls not in result.errorClasses: 

151 storage = getattr(result, storage_attr, []) 

152 setattr(result, storage_attr, storage) 

153 result.errorClasses[cls] = (storage, label, isfail) 

154 

155 def patchResult(self, result): 

156 result.printLabel = print_label_patch(result) 

157 result._orig_addError, result.addError = \ 

158 result.addError, add_error_patch(result) 

159 result._orig_wasSuccessful, result.wasSuccessful = \ 

160 result.wasSuccessful, wassuccessful_patch(result) 

161 if hasattr(result, 'printErrors'): 

162 result._orig_printErrors, result.printErrors = \ 

163 result.printErrors, print_errors_patch(result) 

164 if hasattr(result, 'addSkip'): 

165 result._orig_addSkip, result.addSkip = \ 

166 result.addSkip, add_skip_patch(result) 

167 result.errorClasses = {} 

168

170def add_error_patch(result): 

171 """Create a new addError method to patch into a result instance 

172 that recognizes the errorClasses attribute and deals with 

173 errorclasses correctly. 

174 """ 

175 return make_instancemethod(TextTestResult.addError, result)
