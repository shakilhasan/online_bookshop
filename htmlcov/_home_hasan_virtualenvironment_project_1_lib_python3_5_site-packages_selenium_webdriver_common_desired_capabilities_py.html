23class DesiredCapabilities(object): 

24 """ 

25 Set of default supported desired capabilities. 

26 

27 Use this as a starting point for creating a desired capabilities object for 

28 requesting remote webdrivers for connecting to selenium server or selenium grid. 

29 

30 Usage Example:: 

31 

32 from selenium import webdriver 

33 

34 selenium_grid_url = "http://198.0.0.1:4444/wd/hub" 

35 

36 # Create a desired capabilities object as a starting point. 

37 capabilities = DesiredCapabilities.FIREFOX.copy() 

38 capabilities['platform'] = "WINDOWS" 

39 capabilities['version'] = "10" 

40 

41 # Instantiate an instance of Remote WebDriver with the desired capabilities. 

42 driver = webdriver.Remote(desired_capabilities=capabilities, 

43 command_executor=selenium_grid_url) 

44 

45 Note: Always use '.copy()' on the DesiredCapabilities object to avoid the side
